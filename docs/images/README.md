# 截图说明

请在此目录放置以下截图，以完善 README.md 的效果展示：

## 需要的截图

### 1. main-interface.png
**内容**：主界面/上传界面截图
- 展示初始上传界面
- 显示三种上传方式的提示文字
- 建议尺寸：1200x800 或更高

### 2. result-display.png
**内容**：识别结果展示截图
- 左侧显示原图和边界框
- 右侧显示识别结果（纯文本或详细模式）
- 建议使用有代表性的测试图片
- 建议尺寸：1200x800 或更高

### 3. highlight-demo.png
**内容**：边界框高亮效果截图
- 鼠标悬停在详细结果表格某一行时的效果
- 左侧图片对应区域应显示蓝色高亮
- 建议尺寸：1200x800 或更高

## 截图建议

- 使用清晰的测试图片（如包含中英文的文档、海报等）
- 确保截图清晰，文字可读
- 建议使用 PNG 格式保持清晰度
- 可以使用浏览器的全屏截图工具或专业截图软件

## 如何截图

### Windows
- 使用 `Win + Shift + S` 打开截图工具
- 或使用浏览器开发者工具的设备模拟功能

### macOS
- 使用 `Cmd + Shift + 4` 截取区域
- 或使用 `Cmd + Shift + 5` 打开截图工具

### 浏览器
- F12 打开开发者工具
- Ctrl/Cmd + Shift + P 输入 "screenshot"
- 选择 "Capture full size screenshot"
